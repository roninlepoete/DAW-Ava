# MASTER — Mastering automatique

> Finaliser un mix audio avec loudness, EQ, stereo width et dynamic range optimises.

---

## Outils disponibles

| Outil | Acces | Prix | Forces |
|-------|-------|------|--------|
| **Dolby.io Music Mastering** | API REST | $0.05/min, 200 min gratuites/mois | Metriques retournees (loudness LUFS, EQ 16 bandes, stereo width) |
| LANDR Mastering | API REST B2B + Plugin VST | Sur devis (API) / abonnement (plugin) | Reference mastering IA, 3 niveaux loudness |
| eMastered | Web uniquement | Abonnement mensuel | Cree par ingenieurs Grammy |
| iZotope Ozone 12 | VST/AAX/AU | ~$499 (Advanced) | Mastering dans le DAW |

## Choix DAW-Ava

**Dolby.io** — API la moins chere, metriques detaillees, 200 min/mois gratuites. Voir `specs/MASTER/dolby-io/`.

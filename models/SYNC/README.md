# SYNC — Synchronisation audio-video

> Generer une ambiance sonore synchronisee sur une video.

---

## Outils disponibles

| Outil | Fournisseur | Acces | Forces |
|-------|-------------|-------|--------|
| **MMAudio V2** | Fal.ai | `fal-ai/mmaudio-v2` | Video + texte → audio synchronise |
| MMAudio V2 T2A | Fal.ai | `fal-ai/mmaudio-v2/text-to-audio` | Texte seul → audio |

## Choix DAW-Ava

**MMAudio V2** — pont direct avec Remotion/Mikhaelize. Voir `specs/SYNC/fal-ai/mmaudio-v2/`.
